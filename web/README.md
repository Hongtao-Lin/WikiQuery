Backend:

Database:  
MongoDB  

Run:
use virtualenv  
```bash  
$ virtualenv venv  
$ source venv/bin/activate  
$ pip install -r requirenments.txt

$ python run.py  
```  

JavaScript:  
Change js file,need node and webpack  

```bash  
$ cd app/static/js  
$ npm install  
$ npm install -g webpack  
$ webpack --watch
```
